Instal·lació de Comeet

Per instal·lar el comeet en Ubuntu, primer actualitzi el seu llista de paquets:

sudo apt update

després clona el repositori en el teu servidor

git clone https://github.com/Yaisan/comeet.git

accés a l'repositori clonat

cd comeet

concedeixi permisos per a executar l'script

chmod 0755 comeet.sh

executar l'script amb permisos de root

sudo bash comeet.sh

quan s'executi l'script, li demanarà que ingressi un domini per al servidor

· · · · · · · ·

Set a domain

i l'script estarà acabat i comeet ja estarà instal·lat
