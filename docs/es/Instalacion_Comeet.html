Instalación de Comeet

Para instalar el comeet en Ubuntu, primero actualice su lista de paquetes:

sudo apt update

luego clona el repositorio en tu servidor

git clone https://github.com/Yaisan/comeet.git

acceso al repositorio clonado

cd comeet

conceda permisos para ejecutar el script

chmod 0755 comeet.sh

ejecutar el script con permisos de superusuario

sudo bash comeet.sh

cuando se ejecute el script, le pedirá que ingrese un dominio para el servidor

· · · · · · · ·

Set a domain

y el script estará terminado y comeet ya estará instalado
